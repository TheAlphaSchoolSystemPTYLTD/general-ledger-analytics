**getReportingCode**
----
  Returns an array of structured Reporting Codes for one Account Code in JSON format.
  
* **Version History:**

  TASS v52.5 - Method Added

* **Version:**

  3

* **Permission:**

  General Ledger > Accounts

* **Method:**

  `GET | POST`
  
*  **Params:**

   **Required:**

   `acct_code [string]` -  Account Code
   
   **Optional:**

   none

   **Conditional:**
 
   none
